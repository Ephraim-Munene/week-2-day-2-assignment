# JavaScript Fundamentals Tasks

This repository contains my solutions to three core JavaScript programming exercises. Each script focuses on clean logic, error validation, and standard manual iteration techniques.

## Included Tasks

1. **Task 1: Tip Calculator (`calculateTip`)**
   * Calculates total bill, tips, and splits costs per person based on service quality.
2. **Task 2: Weighted Grade Calculator (`calculateGrade`)**
   * Computes weighted grades from an array of score objects and validates total weights.
3. **Task 3: Password Strength Checker (`checkPasswordStrength`)**
   * Validates passwords based on length, casing, numbers, and special characters.
